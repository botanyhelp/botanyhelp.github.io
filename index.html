Ruby

  • Herokucarriers. Ruby on Rails web app writing to Postgres database all on Heroku (code on github)

  • QRCode. This ruby code creates png formatted images of the QR codes. It also dumps text which can be used to create an HTML/CSS version of the QR code. After running the ruby program with the included example data, six new 300x300 PNG-formatted images will be created in the current directory. The output HTML from the python can be dropped right into any HTML file and will display the QR codes as HTML. The workflow requires python and the ruby requires some modules that are probably not present. Install them with gem like this and run the whole thing like this:

gem install rqrcode
gem install rqrcode_png
ruby qrHTMLandPNGmaker.rb >> rb.txt
python qrHTMLmaker.py >> qr_codes.html
